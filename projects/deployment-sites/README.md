## Metodologia

### Ciclo de Desenvolvimento

1. **Descoberta** → Entender identidade, persona e objetivos de negócio
2. **Prototipação** → Vibe Coding com Lovable — MVP funcional em horas
3. **Refinamento** → Design iterativo com feedback visual contínuo
4. **Validação** → Testes com usuário reais e ajustes
5. **Deployment** → Publicação com monitoramento

### Filosofia de Design
- **Profundidade visual**: assets com camadas e storytelling
- **Motion significativo**: animação como parte da narrativa
- **Hierarquia clara**: tipografia e espaçamento como elementos de identidade
