# 🎬 Amarella Creative — Site Premium Boutique Editorial

**Stack:** TanStack Start · React 19 · Framer Motion · Tailwind v4 · Lenis · Cloudflare Workers · Bun

---

## 🎯 Sobre

Site premium institucional para estúdio de **branding, estratégia, conteúdo e storytelling**. Construído com arquitetura moderna (**TanStack Start** + **React 19 SSR** + **Cloudflare Workers**), animações cinematográficas com **Framer Motion 12** e scroll suave **Lenis**.

**6.655 linhas** · **13 seções** · **3 rotas** · **16 imagens** · **Sistema de design extraído como skill reutilizável**

> 💼 **Projeto para cliente real** — estúdio boutique de vídeo maker e branding.

---

## 📋 Seções

| Seção | Destaque |
|---|---|
| **🏠 Hero** | Full-bleed parallax, h1 com palavra âmbar em itálico, CTAs duplas, tagline script |
| **👥 Sobre** | Duas fundadoras, foto + TiltCard com logo animado |
| **💡 Serviços** | Lista numerada (01–07): Branding, Estratégia, Social Media, Direção Criativa, Cobertura de Eventos, Storytelling, Coprodução Digital |
| **🌱 Filosofia** | 4 pilares visuais com grid de cards |
| **📝 Processo** | Seção escura com as 7 etapas: Pesquisa → Roteiro → Direção → Captação → Edição → Publicação → Consistência |
| **🎬 Bastidores** | Parallax com citação + foto de fundadora filmando |
| **📖 Histórias** | 6 cards de projetos com hover zoom |
| **💬 Depoimentos** | 3 citações de clientes reais |
| **🔗 Links** | Página estilo Linktr.ee com WhatsApp featured + Instagram/LinkedIn/Site/Email |
| **📄 SEO** | JSON-LD LocalBusiness, OpenGraph, sitemap.xml server-side, per-route head() |

## 🎨 Sistema de Design

| Elemento | Detalhe |
|---|---|
| **Paleta** | `oklch` — âmbar quente, creme, bege, ink (charcoal escuro) |
| **Tipografia** | Fraunces (serif) + Inter (corpo) + Caveat (script) |
| **Grid** | 12 colunas editorial com margens generosas |
| **Mood** | Calmo, editorial, artesanal — design boutique premium |
| **Animações** | Framer Motion: Reveal, Stagger, parallax, TiltCard, AnimatePresence, hover transitions |
| **Scroll** | Lenis smooth scroll |

## 🛠️ Stack

```
Framework:    TanStack Start (React 19 SSR + Vite)
Router:       TanStack Router (file-based routes)
Animações:    Framer Motion 12
Scroll:       @studio-freight/lenis
Estilo:       Tailwind v4 + tw-animate-css
Paleta:       oklch (custom theme inline)
Fontes:       Google Fonts via head().links
Deploy:       Cloudflare Workers (wrangler.jsonc)
Pacotes:      Bun
```

## 🤖 Skill Extraída

O sistema de design foi extraído como **skill reutilizável do Hermes Agent** (`editorial-boutique-site`), com referências completas de tokens, receitas de seção, padrões de motion e checklist SEO — permitindo aplicar o mesmo template editorial para outras marcas.
