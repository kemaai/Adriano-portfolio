# 🎨 Estudos de Design & Referências

**Stack:** HTML/CSS · GSAP · Motion Design · Apple-like Design Principles

---

## Sobre

Laboratório particular de estudos de design de alta referência — análise, clonagem e adaptação de interfaces premium para estudo de estrutura, motion, storytelling e sistemas de design.

## Abordagem

> **Nota:** Todos os estudos são para fins de aprendizado. Assets e textos proprietários não são reutilizados. Clones de estudo usam subdomínios temporários com noindex.

### O que é estudado

| Aspecto | Referências |
|---|---|
| **Estrutura** | Hierarquia visual, grids, layouts responsivos |
| **Motion** | GSAP, scroll triggers, micro-interações |
| **Storytelling** | Narrativa visual, progressão de página |
| **Sistemas de Design** | Paletas, tipografia, componentes |

### Principais Referências

- Apple — design principles adaptados
- Linear, Stripe, Vercel — design systems modernos
- Dribbble, Awwwards — tendências visuais
