## Metodologia

### AI-First Development Workflow
1. **Briefing** → Definição de identidade visual e persona da marca
2. **Prototipação rápida** → MVP em Lovable em horas/dias
3. **Refinamento** → Design iterativo com feedback visual
4. **Deployment** → Entrega contínua com validação de qualidade

### Apple-like Design Principles
- Profundidade, não cards/grids planos
- Motion como parte da narrativa
- Hierarquia visual com espaçamento generoso
- Tipografia como elemento de identidade
